# sort-detective-research
Research being conducted on the Sort Detective black-box style application for a CS 301 assignment at CSU East Bay.

General Premise:

- Create a Python script to quickly generate datasets (in .txt format)

- Run all algorithms on each dataset in order 

- Deduce fast/slow algorithms into two groups 

- Further deduce each specific sort algorithm using key characteristics

This README will be updated post-finals.
